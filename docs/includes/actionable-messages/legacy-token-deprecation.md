> [!IMPORTANT]
>Actionable Messages now use Microsoft Entra ID–based token authentication. The transition from legacy (EAT) authentication was completed on June 8, 2026, and legacy authentication is no longer supported.

>Integrations that have not been updated to use Microsoft Entra ID tokens will no longer function. Partners must use Entra ID–based token authentication for all Actionable Messages integrations. [Enable Microsoft Entra ID token for Actionable Messages](../../actionable-messages/enable-entra-token-for-actionable-messages.md).
